# 项目完成说明

## 📋 项目概述

本项目是一个完整的本地部署选股系统，可以调用 AKshare 接口获取 A 股数据，并根据三个技术条件自动筛选符合条件的股票。

**项目名称**: 本地智能选股系统  
**版本**: 1.0  
**完成日期**: 2026年2月26日  
**开发语言**: Python 3.7+

## ✨ 核心功能

### 1. 自动选股 ✓
- 支持检查全部 A 股或指定股票
- 基于三个严格的技术条件筛选
- 所有条件必须同时满足

### 2. 筛选条件 ✓
- **周线金叉**: MA5 > MA10（周线）
- **涨停检测**: 15个交易日内有过涨停（≥9.5%）
- **倍量交易**: 5个交易日内有倍量交易（≥2倍）

### 3. 数据获取 ✓
- 使用 AKshare 接口
- 支持日线和周线数据
- 自动重试机制
- 错误处理完善

### 4. 结果输出 ✓

- 详细日志记录
- 统计信息汇总
- 多种查看方式

### 5. 工具功能 ✓
- 交互式菜单工具
- 深度分析脚本

- 统计分析工具

### 6. 测试支持 ✓
- 完整测试套件
- 单个组件测试

## 📊 选股条件详解

### 条件1: 周线金叉

**定义**: 周线 MA5 > MA10

**计算方式**:
- MA5 = 过去5周收盘价平均
- MA10 = 过去10周收盘价平均
- 当 MA5 > MA10 时判断为金叉

**示例**:
```
周1: MA5 = 10.0, MA10 = 10.5 (无金叉)
周2: MA5 = 10.3, MA10 = 10.3 (金叉形成！)
```

**意义**: 表示股票短期上升趋势开始，是积极的技术信号

### 条件2: 15日涨停

**定义**: 过去15个交易日内至少有一个交易日涨幅 ≥ 9.5%

**检测方式**:
```python
涨幅 = (收盘价 - 前日收盘价) / 前日收盘价
如果 涨幅 ≥ 9.5% 则判断为涨停
```

**示例**:
```
日期      收盘价    涨幅
2026-02-20  10.0   0%
2026-02-21  10.5   5%
2026-02-22  11.0   4.8%
2026-02-23  11.95  8.6%
2026-02-24  13.15  10.0% ← 涨停！（在15日内）
```

**意义**: 表明股票具有强势，吸引市场关注

### 条件3: 5日倍量

**定义**: 过去5个交易日内至少有一个交易日成交量 ≥ 前20日平均 × 2

**检测方式**:
```python
前20日平均成交量 = sum(过去20日成交量) / 20
倍数 = 当日成交量 / 前20日平均成交量
如果 倍数 ≥ 2.0 则判断为倍量
```

**示例**:
```
前20日平均: 1000万股/天
日期      成交量    倍数
2026-02-20  1100   1.1x
2026-02-21  1050   1.05x
2026-02-22  2200   2.2x ← 倍量！（在5日内）
2026-02-23  900    0.9x
2026-02-24  1000   1.0x
```

**意义**: 表明成交量大幅增加，市场参与度提高

## 💡 三条件组合的意义

| 组合 | 含义 |
|------|------|
| 周线金叉 + 涨停 | 短期趋势向上，有强势表现 |
| 涨停 + 倍量 | 成交量配合，买盘充足 |
| 周线金叉 + 倍量 | 趋势形成，量能充足 |
| 全部三个 | **完美组合** - 短期趋势好 + 近期表现强 + 市场关注度高 |

**字段说明**:

- `日期`: 最新数据的日期
- `最新收盘价`: 最新收盘价格
- `周线金叉`: 是否满足周线金叉条件

- `最大涨幅`: 15日内最大涨幅百分比
- `倍量日期`: 最近倍量的日期（可能有多个）
- `最大倍数`: 5日内最大倍数
- `通过筛选`: 是否通过所有三个条件

## 🔧 配置项说明

在 `config.py` 中可以修改的主要配置:

```python
# 技术条件参数
STOCK_FILTER_CONFIG = {
    'ma5_period': 5,              # MA5周期（改为10表示10周期）
    'ma10_period': 10,            # MA10周期（改为20表示20周期）
    'limit_up_days': 15,          # 检查涨停的天数（改为20表示20日内）
    'volume_multiple_days': 5,    # 检查倍量的天数（改为10表示10日内）
    'volume_multiple_ratio': 2.0, # 倍量比例（改为3.0表示3倍）
    'volume_avg_period': 20,      # 平均成交量周期（改为30表示30日均值）
}

# 涨停判断阈值
LIMIT_UP_THRESHOLD = 0.095  # 9.5%（改为0.10表示10%）

# 网络和重试配置
REQUEST_TIMEOUT = 30        # 请求超时（秒）
MAX_RETRIES = 3             # 最大重试次数
RETRY_DELAY = 1             # 重试延迟（秒）

# 指定检查的股票列表（为空表示检查所有）
STOCK_LIST = []             # 示例: ['000001', '000858', '600000']
```

## ⚙️ 技术细节

### 使用的技术栈

- **数据源**: AKshare（免费股票数据接口）
- **数据处理**: Pandas + NumPy
- **编程语言**: Python 3.7+
- **日志记录**: Python logging 模块
- **数据存储**: CSV 文本格式

### 核心算法

1. **移动平均线 (MA)**
   - 简单移动平均 (SMA)
   - 公式: MA = Σ(Close_i) / N

2. **金叉检测**
   - 条件: MA(5) > MA(10)
   - 判断依据: 短期趋势向上

3. **涨停检测**
   - 条件: 任意一日涨幅 ≥ 9.5%
   - 检查周期: 15个交易日

4. **倍量检测**
   - 条件: 日成交量 ≥ 前期平均 × 2
   - 检查周期: 5个交易日

### 数据处理流程

```
获取所有股票代码
    ↓
循环每个股票:
  ├→ 获取日线数据 (120天)
  ├→ 获取周线数据 (52周)
  ├→ 计算MA指标
  ├→ 检查三个条件
  └→ 记录结果
    ↓
保存结果到 CSV
    ↓
输出统计信息
```
